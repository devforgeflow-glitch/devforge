# DevForge - Design System

Bem-vindo ao Storybook do **DevForge**, um template enterprise-grade para desenvolvimento de aplicacoes SaaS modernas.

## Sobre o Design System

O DevForge utiliza um sistema de design consistente baseado em:

- **Tailwind CSS** para estilos utilitarios
- **CSS Variables** para temas dinamicos (light/dark)
- **CVA (Class Variance Authority)** para variantes de componentes

## Cores da Marca

| Cor | Uso | Valor |
|-----|-----|-------|
| **Primary** | Acoes principais, CTAs | Indigo (#6366F1) |
| **Secondary** | Acoes secundarias | Cyan (#06B6D4) |
| **Accent** | Destaques | Amber (#F59E0B) |
| **Destructive** | Acoes destrutivas | Red (#EF4444) |

## Padroes de Botoes

Todos os botoes do DevForge usam formato **pilula** (`rounded-full`), seguindo o guia de estilo definido.

## Dark Mode

O sistema suporta tres modos de tema:

1. **Light** - Tema claro (padrao)
2. **Dark** - Tema escuro
3. **System** - Segue a preferencia do sistema operacional

Use o seletor de backgrounds do Storybook para visualizar componentes em ambos os temas.

## Componentes Disponiveis

### UI Base
- **Button** - Botao com multiplas variantes e tamanhos
- **ThemeToggle** - Alternador de tema light/dark

### Em desenvolvimento
- Input
- Card
- Modal
- Badge
- Skeleton
